ยินดีต้อนรับสู่ API ของฉัน

คำอธิบายสำหรับ /account:

  • /account - ดู account ทั้งหมด
  • /acount/{id} - ดูข้อมูลของ ID
  • put /update/{id} - อัพเดทข้อมูลใน ID
  • delete /delete/{id} - ลบข้อมูล ID

คำอธิบายสำหรับ /login:

  • post /login - ใช้เพื่อล็อคอิน(request(username,password))

คำอธิบายสำหรับ /signup:

  • post /signup - request(email, username, password)
